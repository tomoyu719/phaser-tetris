# Phaser Tetris

A Tetris clone built with Phaser 3 and TypeScript.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev      # Start dev server (port 5173)
npm run build    # Production build
npm run test     # Run unit tests
```

## Controls

| Key | Action |
|-----|--------|
| Left/Right | Move |
| Down | Soft drop |
| Up / Z | Rotate left |
| X | Rotate right |
| Space | Start / Retry |

## Tech Stack

- Phaser 3
- TypeScript
- Vite
- Jest (unit tests)
- Playwright (E2E tests)
